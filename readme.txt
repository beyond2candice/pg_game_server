1. 首先，创建一个Cloudflare帐户：网站链接
2. 在 Hostinger 上购买 VPS（最低 KV2）：链接到购买
3. 购买域名。 （如果您已经有，请继续。）
4. 安装VPS-Ubuntu 20.04。巴西服务器
5. 将域名指向 Cloudflare。您将使用这两个地址
    •hugh.ns.cloudflare.com
    • Paris.ns.cloudflare.com
6.  指出域名后，我们来配置DNS。他们是以下3种
    备注： 
        类型：A
        名称： API
        内容：你的vps号码

        类型：A
        名称： 输入您的域名
        内容：你的vps号码

        类型：Cname
        名称：www
        内容：输入您的域名
7. 下载 Putty 应用程序：下载链接
    ● • 选择与您的PC架构兼容的版本并下载。
    ● 打开putty，输入你的vps ip，点击打开
    ● 以root身份登录并输入您注册的密码在托管商处
